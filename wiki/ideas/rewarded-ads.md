# Idea: Rewarded Ads

Last updated: 2026-05-17

## Concept

An optional, non-intrusive ad mechanic:

1. User voluntarily watches a short ad
2. Earns credits for doing so
3. Credits accumulate and can be redeemed to unlock Pro features (temporarily or permanently)

## Why It's Interesting

- Gives price-sensitive users (especially India) a zero-cost path to Pro
- Fully opt-in — no coercion, no ads in the core flow
- Aligns with the non-goals: "No ads in core flow" is preserved since this is user-triggered

## Constraints (if implemented)

- Must be fully optional — never shown unless user initiates
- Transparent: clearly explain how many credits an ad earns and what they unlock
- Must not interrupt expense-adding or any primary flow

## Status

**Not committed.** This is a captured idea. Implementation decision deferred indefinitely.
